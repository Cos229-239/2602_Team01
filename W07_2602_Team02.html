How to move from a fork to the class repo:
  • 1. Clone the class repo (org repo) if you have not already.
  • 2. In that clone, add your fork as a remote (for example: git remote add fork <your-fork-URL>).
  • 3. Fetch your fork (for example: git fetch fork).
  • 4. Create a new personal branch from dev in the class repo (for example: git checkout -b YourNamePersonal origin/dev).
  • 5. Merge the work from your fork branch into your new personal branch (for example: git merge fork/<your-branch> and resolve any conflicts).
  • 6. Push your personal branch back to the class repo (for example: git push origin YourNamePersonal).
  • 7. From now on, work only in the class repo (on your personal branch + dev); do not keep committing to the fork.
